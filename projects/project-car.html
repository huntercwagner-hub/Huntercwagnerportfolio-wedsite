Automated Wall-Avoiding Car

University of West Florida | EML 4550 Mechatronics Systems

Project Overview

Objective: Design and construct an autonomous car capable of navigating through a hallway or obstacle course while automatically avoiding walls and objects. The system utilized infrared sensors for distance detection and an Arduino microcontroller for real-time decision making and motor control.

This was an individual project completed as part of the Mechatronics course. I designed the full circuit layout, wrote the embedded code from scratch, and hand-built the entire system using breadboards, jumper wiring, and a dual-motor drive platform. The final prototype successfully demonstrated wall-avoidance and autonomous path correction during live testing.

Design and Development

The car was constructed using a two-wheel differential drive system powered by DC motors controlled through an L298N motor controller. An Arduino Uno served as the central control unit, receiving inputs from three infrared sensors positioned at the front and sides of the vehicle. The sensors continuously scanned for nearby obstacles and transmitted distance data to the microcontroller for real-time path adjustment.

I created the circuit diagram and implemented the full wiring setup on a breadboard, ensuring stable power distribution and reliable sensor feedback. The project did not require CAD modeling, as the emphasis was on the electronic circuit design and programming logic.

  • Hardware Components: Arduino Uno, L298N motor driver, 2 DC motors, IR distance sensors, breadboards, jumper wires, 9V battery
  • Software Tools: Arduino IDE (C++), MATLAB (for basic data plotting during calibration)
  • Design Features: Real-time obstacle avoidance, forward/reverse logic, independent motor speed control

The final version implemented smoothing filters for sensor data, priority decision layers for directional control, and safety stop logic when obstacles appeared in multiple directions simultaneously.
