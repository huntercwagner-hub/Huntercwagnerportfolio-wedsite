Project Overview

Objective: Design and construct an automated turret system capable of detecting, tracking, and targeting an object autonomously using servo actuation and sensor input. The turret was programmed and built from the ground up using an Arduino microcontroller, custom mechanical components, and embedded control logic.

This project was completed as a solo effort for my Mechatronics course. I was responsible for every stage of development ‚Äî including circuit design, hardware fabrication, embedded coding, testing, and iteration. Over the span of three full design cycles, I redesigned both the mechanical and software systems until achieving consistent and accurate operation.

Design and Development

The first step involved designing the mechanical structure for the turret base and gimbal system. Using SolidWorks, I created a fully parameterized model allowing for easy adjustment of servo positions and axis alignments. The frame and mounting system were fabricated by hand using aluminum brackets, laser-cut acrylic, and 3D printed supports.

The control system was centered around an Arduino Uno microcontroller. The turret was programmed in C++ using the Arduino IDE to process input data from ultrasonic and IR sensors, calculate angle offsets, and command servo positioning for both pitch and yaw control.

  • Software Tools: Arduino IDE (C++), SolidWorks (mechanical design), MATLAB (data testing)
  • Hardware Components: Arduino Uno, SG90 servos, ultrasonic sensors, IR sensor array
  • Design Iterations: 3 total ‚Äî each improving stability, response speed, and mechanical durability

Control Logic and Programming

The turret operates based on sensor feedback loops that continuously update servo positions to maintain a target lock. My code implements angle filtering and PID-style control for smoother servo response and reduced oscillation. I wrote all embedded code manually, ensuring full understanding of timing constraints and sensor-to-actuator coordination.

During testing, I incorporated diagnostic serial output to monitor response delay, detection accuracy, and servo angle changes in real time. Each iteration of code reduced reaction lag and improved target stability, especially under variable lighting conditions or moving targets.

Testing and Results

After three design revisions, the turret achieved smooth and stable operation, successfully tracking targets within its sensor range. The final system demonstrated real-time detection, movement prediction, and accurate servo tracking. Testing validated that the control logic was robust under varying environmental conditions.

This project demonstrated a complete integration of mechanical design, electronic control systems, and software development ‚Äî core skills of a modern mechatronics engineer. It also reinforced the importance of iterative prototyping and debugging in electromechanical design.
